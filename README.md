具体的编程要求请见 互联网应用开发技术-课程大作业要求-2025.pdf
## 主要技术栈
### web 前端
  i. 使用 React、React Router、Ant Design 等架构开发，运用 HTML、CSS、React、React Router、Ant Design 开发前端响应式网站
  
  ii. 使用 WebPack打包管理
### Java 后端
  i. 使用 Spring 框架开发 (Spring	Boot/JPA/Security)，运用 Java 的 Spring Data方式访问数据，开发前后端集成的网站
  
  ii. 使用 Maven工具打包管理

  iii.使用spring security进行鉴权认证
  
### 数据库
i. 使用 MySQL 关系型数据库

ii. MySQL	Workbench 数据库客户端

### 开发工具
i. IntelliJ IDEA	/ VS Code
